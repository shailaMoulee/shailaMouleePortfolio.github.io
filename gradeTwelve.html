/SHOWCASE





College Exploration

Crafted an immersive college exploration website employing CSS selectors, optimizing the user experience through precise styling and layout enhancements. Leveraged CSS selectors to fine-tune the website's elements, ensuring an intuitive and engaging platform for users to discover information about my college preferences.

HireMe! Profile

Designed a captivating HireMe! profile page for a client, utilizing box model properties to create a visually appealing and structured layout. Employed the box model's features to meticulously arrange elements, enhancing the overall presentation and user interaction, effectively showcasing the client's skills and experience to potential employers.

Social Media Page

Crafted a dynamic and responsive social media page for a client, harnessing the power of Flexbox to build a fluid and adaptive layout. Utilized Flexbox properties to seamlessly arrange content elements, ensuring a visually appealing and easily navigable interface that adapts effortlessly across various devices, offering an engaging social networking experience.

Sitti's Hearth: A Palestinian Tale

Developed an interactive adventure game using jQuery, enhancing user engagement through dynamic functionalities and immersive storytelling. Leveraged jQuery's powerful features to create an interactive and captivating gaming experience, incorporating smooth transitions, engaging animations, and intuitive user interactions, captivating players throughout their thrilling journey.

Chat Space

Designed an abstract chat room utilizing the .append() method, providing users with a real-time messaging experience. Implemented jQuery for seamless DOM manipulation, allowing messages to be dynamically added to the chat interface, fostering fluid communication and enhancing the user interaction.

Seasonal Persona

Crafted an engaging character quiz with personalized outcomes, employing conditional statements to tailor the user experience. Integrated conditional logic to assess user responses, dynamically generating personalized results that align with their choices, resulting in a customized and interactive quiz experience.

Flower Bookmark

Developed a dynamic photo gallery where users can contribute by inputting various image addresses. Employed JavaScript loops and the .push() method for efficient array manipulation, ensuring easy navigation through an expanding collection of visuals.

Olive

Designed a mobile application called Olive, a digital hub for education, advocacy, and action in solidarity with the Palestinian struggle. Utilized Figma design tools to create an app prototype with interactive UI, dynamic layouts, and real-time collaboration.

The Edison Story

Developed "The Edison Story," a captivating web project showcasing the vibrant experiences and achievements of Thomas Edison CTE High School. Leveraged HTML, CSS, and JavaScript to craft an engaging platform to reminisce and celebrate cherished memories while embracing the spirit of innovation and excellence.
